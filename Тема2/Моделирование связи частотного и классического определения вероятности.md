# Условие задачи:

Имеется урна, содержащая 12 шаров: 8 красных и 4 синих. Из урны случайным образом выбирают 5 шаров. Событие A заключается в том, что среди выбранных шаров оказывается ровно 3 красных и 2 синих шара.

# Экспериментальная часть:
Проводится серия экспериментов с различным количеством испытаний (от 10 до 1000). 
В каждом эксперименте:
1) Моделируется выборка из 5 шаров из урны
2) Подсчитывается количество случаев, когда выбрано ровно 3 красных и 2 синих шара
3) Вычисляется частотная вероятность как отношение числа успехов к общему числу испытаний

Реализация кодом:
```python
import matplotlib.pyplot as plt
import numpy as np
from random import sample

def random_combination():
    urn = ['R'] * 8 + ['B'] * 4
    return sample(urn, 5)

N_values = np.array(range(10, 1001, 10))
P_A_values = []

for N in N_values:
    N_A = 0
    for _ in range(N):
        comb = random_combination()
        if comb.count('R') == 3 and comb.count('B') == 2:
            N_A += 1
    P_A_values.append(N_A / N)

print(f"Экспериментальная вероятность: {P_A_values[-1]:.4f}")

# Теоретическая вероятность: C(8,3) * C(4,2) / C(12,5)
P_theoretical = (56 * 6) / 792
print(f"Теоретическая вероятность: {P_theoretical:.4f}")

# Построение графика
plt.figure(figsize=(10, 6))
plt.plot(N_values, P_A_values, label="Частотная вероятность", linewidth=2)
plt.axhline(y=P_theoretical, color='red', linestyle='--', label='Теоретическая вероятность')
plt.xlabel('Количество испытаний (N)')
plt.ylabel('Вероятность P(A)')
plt.title('Сходимость частотной вероятности к теоретической\n(3 красных и 2 синих шара из 12)')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

# Дополнительная информация
print(f"\nДетали расчета:")
print(f"C(8,3) = 56 способов выбрать 3 красных из 8")
print(f"C(4,2) = 6 способов выбрать 2 синих из 4")
print(f"C(12,5) = 792 способа выбрать 5 шаров из 12")
print(f"P = (56 × 6) / 792 = {P_theoretical:.4f}")
```
