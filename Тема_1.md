# Задание:
Представьте, что вы играете в дартс с закрытыми глазами, пытаясь попасть в мишень. Ваши дротики случайным образом разлетаются по квадратной стене, на которой расположены два разноцветных круга.
Смоделируем процесс метания дротиков, где площадь стены в два раза больше площади большого красного круга, а площадь стены в три раза больше площади малого синего круга. Нужно определить, сколько дротиков попадет в кольцевую область между этими двумя кругами.

**Теоретический расчет:**
- Площадь стены: S_стены = side²
- Площадь красного круга: S_красного = πR1² = S_стены/2
- Площадь синего круга: S_синего = πR2² = S_стены/3
- Площадь кольца: S_кольца = S_красного - S_синего = S_стены/2 - S_стены/3 = S_стены/6
- Вероятность попадания: P = S_кольца / S_стены = 1/6 ≈ 0.167

Теоретическая вероятность попадания в кольцо составляет примерно 16.7%.

Реализация кодом: 

```python
import matplotlib.pyplot as plt
from matplotlib.patches import Circle, Rectangle
import numpy as np

def darts(size):
    x0, y0, R = 6, 6, 5
    side = np.sqrt(2 * np.pi * R ** 2)
    R1 = side / np.sqrt(2 * np.pi)
    R2 = side / np.sqrt(3 * np.pi)
    
    fig, ax = plt.subplots(figsize=(5, 5))
    ax = plt.gca()
    ax.add_patch(Rectangle((0, 0), side, side))
    ax.add_patch(Circle((x0, y0), R1, facecolor='gray'))
    ax.add_patch(Circle((x0, y0), R2, facecolor='pink'))
    
    X = np.random.uniform(0, side, size)
    Y = np.random.uniform(0, side, size)
    plt.scatter(X, Y, marker='x', c='black')
    
    distances = (X-x0)**2 + (Y-y0)**2
    inside = len([d for d in distances if R2**2 <= d <= R1**2])
    
    plt.show()
    print(f'В кольцо из {size} дротиков попало {inside}. Доля попавших составила {inside/size:.3f}')

darts(100)
```
